# Power BI Dashboard

This folder contains the Power BI dashboard and screenshots for the Retail Sales Analytics project.

## Contents

- `Retail_Sales_Analytics.pbix` – Main Power BI dashboard file.
- `screenshots/` – Dashboard preview images.

## Dashboard Pages

1. Executive Dashboard
2. Sales Dashboard
3. Customer Dashboard
4. Product Dashboard

These dashboards provide interactive insights into sales performance, customer behavior, product performance, and executive-level KPIs.
