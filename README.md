# 🛍️ Retail Sales Analytics

An end-to-end **Retail Sales Analytics** project built using **Python, Machine Learning, and Power BI** to transform raw retail transaction data into meaningful business insights through data analysis, predictive modeling, and interactive dashboards.

---

## 📌 Project Overview

This project demonstrates the complete data analytics workflow, including:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Advanced Business Analysis
- Feature Engineering
- Machine Learning (Sales & Profit Prediction)
- Interactive Power BI Dashboards
- Version Control using Git & GitHub

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook
- Power BI Desktop
- Git
- GitHub
